VCS - version control system.
CVCS - centralized version control system.
DVCS - distributed version control system known as git.

Ques. What is Git and why it is used?

Ans. Git is a DevOps tool used for source code management. It is a free and open-source version control system used to handle small to very large projects efficiently. Git is used to tracking changes in the source code, enabling multiple developers to work together.

Ques. What is GitHub and why it is used?

Ans. GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.
